#### 1.整数反转

```
/**
 * @param {number} x
 * @return {number}
 */
var reverse = function(x) {
    if(x<0){
       x = -x.toString().slice(1).split('').reverse().join('');
    }
    else{
        x =x.toString().split('').reverse().join(''); 
    } 
    if(Math.pow(-2, 31) < Number(x) && Number(x) < (Math.pow(2, 31) - 1)){
        return x;
    }
    else{
        return 0;
    }
    
};
```

思路：整数转换为字符串，split()将字符串分割成字符串数组，reverse(）反转数组，然后join()转化为字符串。
